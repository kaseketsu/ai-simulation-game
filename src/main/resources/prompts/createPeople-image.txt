【角色定位】
为文字修仙食肆模拟经营游戏生成单张修士顾客的UI级头像/半身像，用于游戏内顾客展示、互动界面，非剧情主角，风格统一、辨识度高，适配2D文字游戏视觉体系。

【输入变量格式参考】
- 修士姓名：{{name}}
- 性别：{{gender_desc}} // 0=男性，1=女性
- 身份：{{status_desc}} // 0=散修，1=内门弟子，2=核心弟子，3=宗门长老
- 地域风格：{{region_desc}} // 0=玄洲，1=沧澜
- 核心性情：{{temperament_desc}} // 0=沉稳，1=高傲，2=节俭，3=热忱，4=凌厉，5=神秘
- 性格特质：{{traits[0]}}、{{traits[1]}}

【实际输入（若无则采用默认值）】
%s

【强调】
生成的图片必须是像素游戏风格，如以撒的结合游戏中 8-bit 像素风
生成的图片上不要有任何文字!!!!

【视觉生成规则（必须严格遵守）】
1. 图片类型与尺寸：
   - 类型：正方形头像/半身像（胸部以上），无全身像
   - 尺寸：512×512像素（适配游戏UI）
   - 构图：居中构图，面部清晰无遮挡，占画面60%以上

2. 风格基础：
   - 整体风格：国风修仙简约立绘，线条清晰流畅，色彩饱和度适中，无厚涂/写实/3D风格
   - 色彩基调：玄洲=水墨风低饱和（青/白/灰为主），沧澜=高对比鲜明（红/金/蓝为主）
   - 背景：极简修仙元素（淡云纹/灵雾/浅木纹理），无复杂场景，避免抢主体
   - 像素风格

3. 身份视觉特征（与status强绑定）：
   - 散修（status=0）：服饰朴素（粗布道袍/短打），无华丽配饰，仅随身小法器（如木簪/粗布包）
   - 内门弟子（status=1）：宗门统一制式道袍（有宗门纹章），配饰简单（玉佩/铜饰）
   - 核心弟子（status=2）：定制道袍（绣纹精致），配饰典雅（玉簪/灵晶吊坠），带道号标识（如发带刻字）
   - 宗门长老（status=3）：华贵道袍（锦缎/暗纹），配饰贵重（灵玉/法印），有岁月感（须发斑白/面容沧桑）

4. 性别视觉特征（与gender强绑定）：
   - 男性（gender=0）：剑眉星目/粗犷质朴（适配身份），无娘化，修仙者气质（无现代发型）
   - 女性（gender=1）：清丽/英气（适配性格），发髻规整（玄洲偏素雅/沧澜偏利落），无过度暴露

5. 性格视觉特征（与temperament+traits强绑定）：
   - 沉稳（temperament=0）：面容平和，眼神温和，嘴角平直，无夸张表情
   - 高傲（temperament=1）：眼神倨傲，头微扬，嘴角微撇，面部线条硬朗
   - 节俭（temperament=2）：衣着无多余装饰，眼神谨慎，手部微攥（细节）
   - 热忱（temperament=3）：面带笑意，眼神明亮，肢体放松（如抬手打招呼）
   - 凌厉（temperament=4）：眼神锐利，眉峰微挑，面部线条棱角分明
   - 神秘（temperament=5）：半遮面（如薄纱/斗笠檐），眼神深邃，服饰有暗纹/隐纹

【禁忌要求（严格禁止）】
1. 禁止出现现代元素（手机/文字/服饰）、血腥暴力、过度商业化装饰
2. 禁止复杂背景（如宫殿/战场），仅保留极简修仙氛围
3. 禁止面部模糊/遮挡（神秘性格仅允许浅薄纱，不遮挡五官）
4. 禁止偏离修仙风格（如古风武侠/玄幻魔幻）
5. 禁止重复同款造型（即使属性相似，也需调整服饰细节/发型）

【输出要求】
- 格式：PNG/JPG，透明背景（优先）
- 清晰度：高清无噪点，边缘无锯齿
- 一致性：视觉风格与游戏内其他修士立绘统一，无违和感
